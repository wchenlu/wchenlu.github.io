华为VRP系统

一、什么是VRP

VRP是华为公司数据通信产品的通用操作平台。

VRP的功能：

  • 实现统一的用户界面和管理界面
  • 实现控制平台功能，并定义转发平台接口规范
  • 实现产品转发平面与VRP控制平面之间的交互
  • 屏蔽各产品数据链路层对于网络层的差异

文件系统：

文件系统对于很多设备都很重要；文件系统是指对存储器中文件、目录的管理。

  • 系统软件

    系统软件是设备驱动运行的必备软件，常见的后缀名为(.cc)。

  • 配置文件

    配置文件是用户配置命令保存的文件，常见的后缀名有(.cfg , .zip , .dat)。

  • 补丁文件

    补丁是一种与设备系统软件兼容的软件，常见文件后缀名为（.pat）。

  • PAF 文件

    PAF文件是根据用户对产品需求提供了一个简单有效的方式来裁剪产品的资源占用和功能特性。常见的文件后缀名(.bin)

存储设备

  • SDRAM

    同步动态随机存储器

  • flash

    属于非易失性存储器

  • NVRAM

    非易失随机读写存储器

  • SD Card

    断电后，不会丢失数据，存储容量较大，一般出现在主控板上

  • USB

    接口，用于外接大量存储设备

设备管理

命令行和web网管是常见管理方式

  • web网管方式

    通过图形化操作界面，对设备的管理维护比较直观，但是只能实现部分功能的管理与维护。这个维护方式可以通过HTTP和HTPS的方式登录设备。

  • 命令行方式

    需要用户使用设备提供的命令行对设备进行管理与维护，此方式可实现对设备的精细化管理，但是要求用户熟悉命令行。命令行可通过Console口、Telnet或者SSH的方式登录设备。

命令行登录方式

  1. 本地登录：

    • 因为设备在第一次上电时没有被配置过，所以不能通不过Telnet或者SSH远程登录；这个情况下，就可以用一种特殊的线缆（Console线缆：这种线连接在设备的位置是一个RJ45的网线水晶头连接PC的是一个Console的头，这个头可能大多笔记本都没有，就需要一个USB转Console的设备）进行主机与设备之间的连接，连接之后通过一个名为puTTY的软件进行一些对应的参数设置，那么就可以将设备连接进网。

  2. 远程登录：

  • ​ SSH登录

​ 加密远程登录，相对比较安全。登录是选择该模式则默认22号端口。

  • ​ Telnet登录

​ 普通登录，有被泄露的风险，这个模式端口默认为23号。

无论哪种登录方式，主机与设备必须有连接，必须ping的通

VRP用户界面

  • Console用户界面
    1. 这个界面用来管理和监控通过Console口登录的用户
    2. 用户终端的串行口可以与设备Console直接连接，实现对设备的本地访问。
  • VTY用户界面
    1. VTY用户用来管理和监控通过VTY方式登录的用户
    2. 用户通过终端与设备建立Telnet连接后，就是建立了一条VTY通道，通过VTY通道实现对设备的远程访问。
    3. 需要注意如果使用Telnet远程登录，需要事先对VTY界面做一些配置

VRP用户级别

VRP可以实现不同级别的用户能够执行不同级别的命令。限制不同用户对设备的操作

用户等级 命令等级 名称 说明
0 0 参观级 可使用网络诊断工具命令（ping 、 tracert）、从本设备出发访问外部设备的命令（Telnet客户端命令）、部分display命令等。
1 0 and 1 监控级 用于系统维护，可使用display等命令。
2 0，1 and 2 配置级 可使用业务配置命令，包括路由、各个网络层次的命令，向用户直接提供网络服务。
3-15 0，1，2 and 3 管理级 0可使用应用于系统基本运行的命令，对业务提供支撑作用，包括文件系统、FTP、TFTP下载、命令级别设置命令以及用于业务故障诊断的debugging命令等。

命令行界面

上述登录成功后即可进入命令行界面CLI（Command Line interface）

其中，尖括号代表用户视图，而方括号代表系统视图。两者之间可以进行的操作不同。

命令行基础

基本命令结构

命令字+关键字+参数（参数包括参数名和参数值）

例如：display ip interface GE0/0，查看借口信息的命令；其中：

命令字 display
关键字 ip
参数名 interface
参数值 GE0/0

命令行视图

视图，可以理解为模式；设备提供了多样的查询和命令，为便于玉狐使用这些命令，VRP系统按功能分类将命令分别注册在不同的命令视图下。

undo命令行

在命令行前面加上undo关键字，即为undo命令行；这个命令行主要是用来恢复缺省、禁用某个功能或者删除某项配置。

命令快捷键

这里说几个比较实用的

Ctrl+Z：返回用户视图

Ctrl+C：终止当前命令运行

Ctrl+】：终止当前连接或者切换连接

常见文件系统操作命令

命令 作用
delete 删除文件，放进回收站
undelete 恢复删除文件
reset recycle-bin 彻底删除回收站中的文件，注意这个不可恢复
pwd 查看当前目录
dir 显示当前目录下的文件信息
more 查看文本文件的具体内容
cd 修改用户界面的工作目录
mkdir mk是make的缩写意思是创造，这个命令是创建新的目录
rmdir rm是remove的缩写，这个命令的意思是删除
copy 复制文件
move 移动文件，注意这几个是关键字，后面需要参数
rename 对文件重命名

基本配置命令

  1. 配置设备名称

[Huawei]sysname name

  1. 设置系统时钟

    [Huawei]clock timezone time-zone-name{add|minus}offset

    用来对本时区信息进行设置

    clock datetime[utc] HH:MM YYY-MM-DD

    用来设置设备当前或UTC日期和时间

    clock daylight-saving-time

    用来设置设备的夏令时

  2. 配置命令等级

[Huawei]command-privilege level level view view-name command-key

用来指定视图内命令的级别。命令的级别分别为参观、监控、配置和管理四个级别，分别对应标识0-3

  1. 配置用户是否通过password登录

    [Huawei]user-interface vty 0 4

    [Huawei-ui-vty0-4]set authentication password cipher infomation

    用来进入指定的用户视图并配置用户认证方式为password。系统支持的用户界面包括Console用户界面和VTY用户界面，Console界面用于本地登录，VTY界面用于远程登录，默认情况下，设备组多支持十五个用户同时通过VTY访问。

  2. 配置用户界面参数

    [Huawei]idlle-timeout minutes[seconds]

    设置用户界面断开连接超时时间，如果用户一段时间没有输入命令，系统将断开连接，默认时长十分钟。

  3. 给接口配置ip地址

[Huawei]interface interface -number

[Huawei-interface-number]ip address ip address

用来给设备上的物理或者逻辑接口配置ip地址。

  1. 查看当前运行的配置文件

    [Huawei]display current-configuration

    如果配置文件不保存，则重启后丢失

  2. 配置文件保存

Save

  1. 查看保存的配置

    display saved-configuration

  2. 清除已经保存的配置

    reset saved-configuration

  3. 查看系统启动配置参数

    display statup

    查看设备本次及下次启动相关的系统软件、备份系统软件、配置文件、Licence文件、补丁文件及语音文件。

  4. 配置系统下次启动时使用的配置文件

<Huawei>**startup saved-configuration** *configuration-file*

设备升级时，可以通过此命令让设备下次启动时加载指定的配置文件

  1. 配置设备重启

    reboot

注意reboot、startup、reset等对文件的操作命令都是在用户模式下配置，但dispaly一般限制比较少，可以在多个视图下使用。
